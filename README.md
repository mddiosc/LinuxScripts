# Utilidad de gestión de LAMP

Creado por:

* Luis Martin
* Miguel Ángel Riera
* Miguel Angel de Dios

## Descripción

Esta utilidad es un conjunto de scripts que simplifican la instalación y configuración de un entorno LAMP.

## Módulos

* Instalación de Apache
* Instalación de MySQL
* Instalación de PHP
* Gestión de certificados
* Configuracion de hosts virtuales en Apache
* Gestion bases de datos MySQL

## Requerimientos

* Ubuntu 14 o superior
* Bash 4
* El usuario debe tener permisos de **sudo**, o ser **root** para el correcto funcionamiento de estos scripts.
