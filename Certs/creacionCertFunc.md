# Script de creación de un CSR (Petición de firma de certificado)

## Autor : Miguel Ángel de Dios

Este script genera la clave privada (private key), y el CSR, el cual se utiliza para la posterior generación del certificado seguro. Dicho CSR puede ser enviado a una entidad emisora (como por ejemplo digicert), o ser utilizado para generar un certificado autofirmado una CA propia.

Comandos utilizados:

* echo
* cp
* read
* sed
