# Módulo de gestión de certificados
## Autor: Miguel Ángel de Dios Calles
### Directorio CNF
*******************************************************************

En este directorio se almacenarán los ficheros de configuración (cnf) que serán utilizados durante la gestión de los diferentes apartados del módulo. Todos los ficheros tendran extension .cnf:


| Fichero | Uso
| --- | --- |
| *.key.cnf | Ficheros de generación de claves
| *.CA.cnf  | Ficheros de generación de CA
| *.csr.cnf | Ficheros de generacion de CSR
